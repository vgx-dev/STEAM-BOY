## 1. STEAM-BOY とは

**STEAM-BOY** は、蒸気・水・ドレンなどの流体配管システムを設計・シミュレーションするための Web アプリケーションです。PDF 図面を背景として読み込み、その上に機器や配管を重ねて描くことができます。

### 主な特徴

- **ブラウザだけで動作** — インストール不要。Web ブラウザで開くだけで使えます。
- **PWA 対応** — オフラインでも利用できます。ホーム画面に追加するとアプリのように起動できます。
- **リアルタイムシミュレーション** — 弁の開閉操作に連動して流体の流れが即座に更新されます。
- **操作マニュアル自動生成** — 弁操作の手順書を作成し、印刷・PDF 出力ができます。
- **異常検知** — 孤立ノード・行き止まり・過圧などを自動で検出してアラートを表示します。

## 3. 基本操作

### 3-1. キャンバスの移動（パン）

マウスのホイールをスクロールするか、スクロールバーをドラッグして図面を移動します。

### 3-2. ズーム

| 操作 | 説明 |
|------|------|
| `+` ボタン | 拡大 |
| `−` ボタン | 縮小 |
| ズームスライダー | 任意の倍率（20%〜300%）に変更 |
| ズーム表示（数字部分）をクリック | 100% にリセット |
| `⤢` ボタン（全体表示） | 全ノードが画面に収まるようにフィット |
| `Reset` ボタン | ズームを 100% に戻す |

### 3-3. ツールの切り替え

ツールバーの各ボタンをクリックしてアクティブなツールを切り替えます。現在のツールは青色でハイライト表示されます。

| ツール | アイコン | キー | 説明 |
|--------|----------|------|------|
| 操作・選択 | 矢印 | `V` | ノードの選択・移動・弁操作 |
| 回転 | 回転矢印 | `R` | ノードを回転 |
| Source | S | `S` | 蒸気源を配置 |
| Valve | A | `A` | バルブを配置 |
| Trap | T | `T` | スチームトラップを配置 |
| Drain | D | `D` | ドレンポイントを配置 |
| Split | Y | `Y` | 分岐点を配置 |
| Merge | M | `M` | 合流点を配置 |
| Reducer | E | `E` | 減圧弁を配置 |
| Safety Valve | Q | `Q` | 安全弁を配置 |
| Heat Exchanger | H | `H` | 熱交換器を配置 |
| Pressure Gauge | G | `G` | 圧力計を配置 |
| ピン留め | 📌 | `P` | ノードをマスタ管理に登録/解除 |

### 3-4. ノードの選択

- **単一選択** — 選択ツール（V）でノードをクリック
- **複数選択** — 空白部分をドラッグして矩形選択。または `Shift + クリック` で追加選択
- **選択解除** — `Esc` キー、または選択ツールバーの「✕ 解除」をクリック

### 3-5. ノードの移動

選択ツールでノードをドラッグします。複数選択中は選択したすべてのノードが一緒に動きます。

### 3-6. ノードの削除

ノードを選択して `Delete` キーを押すか、選択ツールバーの「削除」ボタンをクリックします。右クリックメニューからも削除できます。

### 3-7. 右クリックメニュー

ノードや配管を右クリックすると、コンテキストメニューが表示されます。

| メニュー項目 | 説明 |
|-------------|------|
| 削除 | 選択した要素を削除 |
| 名前変更 | ノード名を変更（インライン入力） |
| ハイライト | 連結する経路を強調表示 |

### 3-8. 元に戻す（Undo）

`⌘Z`（Mac）または `Ctrl+Z`（Windows）を押すか、ツールバーの「↩」ボタンをクリックします。画面下部に「元に戻しました」というトースト通知が表示されます。

### 3-9. ゼンモード（作業領域の全画面化）

`Tab` キーを押すと、ツールバー・パネル類がすべて非表示になり、キャンバスだけが表示されます。もう一度 `Tab` を押すと元の表示に戻ります。

> **ヒント:** 資料として画面を共有するときや、印刷・スクリーンショット時に便利です。

## 4. ノードの種類と配置

### 4-1. ノードの配置方法

1. ツールバーから配置したいノードのツールをクリック（またはキーボードショートカット）
2. キャンバスの任意の場所をクリック → ノードが配置されます
3. 配置後は自動的に操作ツール（V）に戻ります

グリッドスナップが有効な場合、ノードはグリッドに沿って配置されます。

### 4-2. ノードの種類

#### Source（蒸気源）— キー: `S`
蒸気を供給する起点です。ここから流体が出発します。設定パネルで供給圧力（MPa）を設定できます。

#### Valve（バルブ）— キー: `A`
流れを制御する弁です。**閉止 / 微開 / 開放** の 3 段階で操作できます。

- **閉止（赤）**: 流れを完全に遮断
- **微開（橙）**: 流量を絞った状態
- **開放（緑）**: 完全に開いた状態

> バルブをクリックするとポップアップが表示され、状態を切り替えられます。

#### Trap（スチームトラップ）— キー: `T`
蒸気を通過させず、ドレン（凝縮水）だけを排出する機器です。

#### Drain（ドレン）— キー: `D`
流体の最終排出点です。ここに流体が到達すると、流体が外部に排出されたとシミュレーションされます。

#### Split（分岐）— キー: `Y`
一つの流れを複数に分岐します。

#### Merge（合流）— キー: `M`
複数の流れを一つに合流します。

#### Reducer（減圧弁）— キー: `E`
上流の圧力を設定した下流圧力に減圧します。クリックするとスライダーが表示され、出口圧力を調整できます。

#### Safety Valve（安全弁）— キー: `Q`
設定圧力を超えた場合に自動的に開いて過圧を防ぐ弁です。マスタ管理で逃がし圧力（MPa）を設定できます。

#### Heat Exchanger（熱交換器）— キー: `H`
蒸気と水の熱交換を行う機器です。最高温度の超過を異常検知します。

#### Pressure Gauge（圧力計）— キー: `G`
配管の圧力を計測・表示するためのノードです。シミュレーション結果の圧力値がラベルに表示されます。

### 4-3. ノードのラベル

各ノードには識別番号（例: `V-001`）が自動的に付与されます。

- **設定パネル「表示」セクション**で、ラベルを常時表示するか、ホバー時のみ表示するかを切り替えられます。
- **名前変更**: 右クリック → 「名前変更」で任意のラベルに変更できます。
- **ラベル位置の調整**: ラベルをドラッグすると位置を調整できます。
- **連番再振**: 設定パネルの「連番再振」ボタンで全ノードのIDを順番に振り直せます。

### 4-4. ノードの回転

回転ツール（R）を選択し、ノードをクリックするたびに 90° ずつ回転します。または操作ツール（V）で回転ツールに切り替えずに回転させるには、ノードを右クリックメニューから操作することもできます。

## 5. 配管の接続

### 5-1. 接続モードの開始

ツールバーの **「配管接続（C）」** ボタンをクリックするか、キーボードの `C` キーを押します。ボタンがオレンジ色に変わり、接続モードがオンになります。

### 5-2. 配管を引く

1. 接続モードで **始点のノード**をクリック（強調表示されます）
2. **終点のノード**をクリック → 配管（エッジ）が作成されます
3. 接続モードは継続するので、続けて別の接続を行えます
4. `Esc` キーか「配管接続」ボタンを再度クリックして接続モードを終了します

### 5-3. 中継点（ウェイポイント）の追加

配管を直線以外の経路で引きたい場合、**配管上をダブルクリック**すると中継点が追加されます。中継点をドラッグして配管の経路を変更できます。

**中継点の削除**: 中継点を右クリックまたはダブルクリックすると削除されます。

### 5-4. 配管の削除

配管を右クリックして「削除」を選択します。または配管を選択して `Delete` キーを押します。

### 5-5. 自動エルボー挿入

設定パネルの「自動エルボー」をオンにすると、配管が水平・垂直方向に自動的に折れ曲がって接続されます。

## 6. 流体シミュレーション

### 6-1. シミュレーションの仕組み

STEAM-BOY では、ノードや配管の接続関係と弁の開閉状態に基づいて、流体の流れをリアルタイムで計算します。

流体の種類は以下の 4 種類です:

| 種類 | 表示色 | 説明 |
|------|--------|------|
| 空（EMPTY） | グレー | 流体が流れていない状態 |
| 水（WATER） | 紫 | 水が流れている状態 |
| ドレン（DRAIN） | 青 | 凝縮水が流れている状態 |
| 蒸気（STEAM） | 赤 | 蒸気が流れている状態 |

### 6-2. 流れのアニメーション

アクティブな配管（流体が流れている配管）には、**白い破線が流れるアニメーション**が表示されます。流れの速度は流量に応じて変化します。逆流している場合は破線が逆方向に流れます。

### 6-3. 残留流体

弁を閉じた後、配管内に残った流体（凝縮水など）は**残留流体**として表示されます。残留状態の配管は、破線が流れずに静止した状態で表示されます。

### 6-4. 密閉空間

すべての出口が閉じられた区間は**密閉空間**として検出され、特別なアニメーション（点滅・パルス）で示されます。密閉空間は過圧の危険性があるため注意が必要です。

### 6-5. 圧力の表示

各ノードには、シミュレーションで計算された圧力値（MPa）がラベルに表示されます（表示設定による）。Pressure Gauge ノードでは特に圧力値が分かりやすく表示されます。

## 7. 弁操作

### 7-1. 弁の開閉

操作ツール（V）でバルブをクリックすると、**バルブ操作ポップアップ**が表示されます。

| ボタン | 状態 | 説明 |
|--------|------|------|
| 閉 | CLOSED | 流れを完全に遮断（赤） |
| 微 | PARTIAL | 流れを絞る（橙） |
| 開 | OPEN | 完全に開放（緑） |

### 7-2. 減圧弁（Reducer）の操作

Reducer ノードをクリックすると、**圧力スライダー**が表示されます。スライダーで出口圧力（MPa）を設定します。設定した圧力以下に減圧されて下流に流れます。

### 7-3. What-if プレビュー

弁の操作ポップアップが表示されている間、それぞれの状態にした場合の流体の変化を**視覚的にプレビュー**できます（下部に変化の警告が表示されます）。

### 7-4. 隔離トレース（Alt + クリック）

ノードを `Alt + クリック`（Mac では `Option + クリック`）すると、そのノードに接続された経路のみを**ハイライト**します。隔離作業時に関連する機器を把握するのに便利です。再度 `Alt + クリック` または `Esc` でハイライトを解除します。

## 8. 異常検知

STEAM-BOY は以下の異常を自動的に検出し、**設定パネルの「異常」セクション**に一覧表示します。また、ミニマップ上でも異常のある箇所が赤くマークされます。

| 異常タイプ | 説明 |
|-----------|------|
| **孤立（isolated）** | 配管が一本も接続されていないノード |
| **行き止まり（deadend）** | 流体が入ってくるが、出口がないノード |
| **過熱（overheat）** | 熱交換器が最高温度設定を超えた状態 |
| **安全弁作動（relief）** | 安全弁が設定圧力を超えて作動している状態 |
| **ドレン蒸気（steam in drain）** | ドレンラインに蒸気が流れている異常な状態 |

異常が検出されると、設定パネルのボタンに赤いバッジが表示されます。クリックして詳細を確認し、該当するノードをクリックすれば図面上でその場所にジャンプできます。

## 9. バルブマトリクス

画面右上に表示される**バルブマトリクス**は、図面内のすべてのバルブの状態を一覧で確認・操作するためのパネルです。

### 9-1. 表示内容

各バルブの行には以下の情報が表示されます:

- **バルブ ID / 名前**
- **現在の状態**（開 / 微 / 閉）— カラーバッジで表示
- **ピン留めボタン**（📌）— マスタ管理への登録
- **操作ボタン**（閉 / 微 / 開）

### 9-2. フィルター

マトリクス上部のフィルターボタンで、表示するバルブを絞り込めます:

| ボタン | 表示内容 |
|--------|----------|
| 全 | すべてのバルブを表示 |
| 開 | 開放中のバルブのみ |
| 微 | 微開中のバルブのみ |
| 閉 | 閉止中のバルブのみ |

### 9-3. 表示密度の変更

右上の「⊞」ボタンをクリックすると、表示を**コンパクト**と**通常**で切り替えられます。

### 9-4. 最小化

パネル上部の「▼」ボタンで最小化（折りたたみ）できます。

## 10. ミニマップ

画面左下の**ミニマップ**は、図面全体の縮小表示です。

- **現在の表示範囲**が枠で示されます
- 異常のあるノードは**赤い点**で表示されます
- **ミニマップ内をクリック**すると、その位置にキャンバスがスクロールします
- 右上の「▼」ボタンで最小化できます

## 11. マスタ管理

**「マスタ管理」ボタン**（画面右下付近）をクリックすると、機器のマスタデータを管理する専用画面が表示されます。

### 11-1. 機器の登録

ピン留め（P キー）したノードがマスタ管理に登録されます。マスタ管理画面では以下の情報を編集できます:

| フィールド | 説明 |
|-----------|------|
| **ID / 名前** | 機器の識別子と名称 |
| **種別** | ノードのタイプ（Valve、Trap など） |
| **最高圧力** | 機器の耐圧上限（MPa） |
| **逃がし圧力** | 安全弁の作動圧力（MPa） |
| **画像 URL** | 機器の写真・図面へのリンク |

### 11-2. 画像ビューア

マスタ管理で画像 URL を登録すると、図面上でそのノードにカーソルを合わせたとき（または選択時）に、**画像ビューア**に機器の画像が表示されます。

### 11-3. タブ切り替え

マスタ管理画面には複数のタブがあります:

- **機器一覧**: 全機器の一覧（名前・種別・圧力設定）
- **バルブ**: バルブのみに絞った一覧

### 11-4. 分割表示

マスタ管理画面を開いたまま図面を確認したい場合は、**「⬌ 左右に分割」** ボタンで画面を分割できます（図面とマスタ管理が並んで表示されます）。

## 12. 操作マニュアル機能

設定パネルの **「📋 操作マニュアル」** ボタンをクリックすると、弁操作の手順書を作成・管理する画面が開きます。

### 12-1. フェーズ構成

マニュアルは複数の**フェーズ**（段階）に分けて構成できます。デフォルトのフェーズ例:

1. **準備** — 作業前の確認事項
2. **暖機** — ウォームアップ手順
3. **確認** — 状態確認
4. **運転** — 通常運転への移行
5. **停止** — 停止手順

フェーズは「＋ フェーズを追加」で追加でき、ドラッグして並べ替えられます。

### 12-2. 手順の追加

各フェーズの「＋ 手順を追加」をクリックして、手順の種類を選択します:

| 手順種別 | 説明 |
|---------|------|
| **バルブ操作** | 特定のバルブを操作する手順（対象バルブ・状態・理由を記載） |
| **確認** | 目視確認などの確認作業（チェックボックス付き） |
| **見出し** | セクションの見出し（手順の区切り） |
| **メモ** | 自由記述のメモや注意事項 |

### 12-3. 手順の編集

各手順行には以下の操作ができます:

- **⠿（ドラッグハンドル）** — ドラッグで並べ替え
- **▲ / ▼** — 上下に移動
- **⧉（複製）** — 手順を複製
- **✕（削除）** — 手順を削除
- **対象バルブの選択** — ドロップダウンで選択、または「🎯 図から選択」ボタンで図面上のバルブをクリック

### 12-4. ライブシミュレーション連携

バルブ操作手順には、シミュレーションで計算された**期待圧力値**が自動表示されます。手順を実行した場合の流体状態が確認できます。

### 12-5. 状態プレビュー

「🎬 状態プレビュー」ボタンをオンにすると、手順行をクリックするだけで**その手順まで実行した弁状態**が図面に一時反映されます（編集内容は保存されません）。

### 12-6. 図から選択

「🖱 図から選択」ボタンをクリックすると、**図面上のバルブをクリックして手順に追加**できます。直接ピックすることで入力ミスを防げます。

### 12-7. 経路順に並べ替え

「🧭 経路順」ボタンで、バルブ手順を**上流から下流の順**に自動並べ替えします。

### 12-8. 一括操作

`Ctrl + クリック`（複数行を選択）した後、以下の一括操作が可能です:

- **✔ 一括チェック** — 選択した確認項目をまとめてチェック
- **⧉ 複製** — 選択した手順をまとめて複製
- **✕ 削除** — 選択した手順をまとめて削除

### 12-9. 印刷・帳票出力

「🖨 印刷設定」ボタンをクリックすると、帳票の体裁を設定できます:

| 設定項目 | 説明 |
|---------|------|
| 用紙サイズ | A4 / レター など |
| ヘッダー情報 | 書類番号・作成者・会社名など |
| 承認欄 | 作成・確認・承認の署名欄の表示/非表示 |
| フッター | ページ番号・注意書きなど |

「👁 プレビュー」で印刷時のイメージを確認し、「印刷」ボタンで出力します。

### 12-10. Undo / Redo

操作マニュアル画面内でも **「↩ 元に戻す（Ctrl+Z）」** と **「↪ やり直し（Ctrl+Y）」** が使えます。

### 12-11. キーボード操作（操作マニュアル内）

| キー | 動作 |
|------|------|
| `Enter`（行末で） | 同種の手順を直下に追加 |
| `Ctrl + ↑ / ↓` | 行を上下に移動 |
| `Ctrl + D` | 行を複製 |

## 14. 共有リンク

設定パネルの **「🔗 共有リンク」** ボタンをクリックすると、現在の図面データを URL にエンコードした共有リンクが生成されます。

- 生成された URL を相手に送るだけで、同じ図面を共有できます。
- 操作マニュアルのデータも含まれます。
- **インターネット接続不要** — データは URL 内に直接埋め込まれます（サーバーへのアップロードは行いません）。

> **注意:** URL が非常に長くなる場合があります。メールやチャットツールでリンクが切断されないよう注意してください。

## 16. データ管理

設定パネルの「データ管理」セクションで、図面データの保存・読み込み・クリアができます。

### 16-1. 自動保存

**自動保存**（デフォルト: ON）が有効の場合、操作のたびにブラウザのローカルストレージにデータが自動的に保存されます。

- ブラウザを閉じて再度開いても、前回の状態から続けられます。
- 自動保存が OFF の場合、画面下部に **「🔒 閲覧モード（自動保存 OFF・変更は保存されません）」** バッジが表示されます。

### 16-2. 手動保存（エクスポート）

「保存」ボタンをクリックすると、図面データが **JSON ファイル**としてダウンロードされます。大切な図面はこまめに手動保存しておくことをお勧めします。

### 16-3. 読み込み（インポート）

「読込」ボタンをクリックして JSON ファイルを選択すると、保存した図面を復元できます。

> **注意:** 読み込みを行うと現在の図面が上書きされます。事前に手動保存することをお勧めします。

### 16-4. クリア

「クリア」ボタンをクリックすると、現在の図面をすべて削除してまっさらな状態に戻ります。**この操作は元に戻せません**のでご注意ください。

### 16-5. 連番再振

「連番再振」ボタンをクリックすると、全ノードの ID を種別ごとに連番（例: V-001, V-002...）で振り直します。

## 18. パフォーマンスについて

STEAM-BOY は大規模な図面でも快適に動作するよう、以下の最適化が自動的に機能します。

### 18-1. 自動最適化モード

| 機能 | 有効になる条件 | 効果 |
|------|--------------|------|
| **ビューポートカリング** | 画面外のエッジが 300 本を超えたとき | 画面外の配管のアニメーションを停止してパフォーマンスを改善 |
| **perf-lite モード** | ノード + エッジが合計 500 を超えたとき | ノードのアニメーション（点滅など）を停止し、蒸気エッジのグラデーションを単色化 |
| **Web Worker 計算** | ノード + エッジが合計 2000 を超えたとき | シミュレーション計算をバックグラウンドスレッドに移してメインスレッドの負荷を軽減 |
| **Canvas フローアニメ** | 常時 | 流れ破線を SVG 個別アニメではなく単一 Canvas に統合描画。多数の配管でも高 FPS を維持 |

これらの最適化は自動的に有効になり、ユーザーが手動で設定する必要はありません。

### 18-2. パフォーマンス計測（上級者向け）

URL に `?perf=1` を追加してページを開くと、FPS や描画時間のリアルタイム計測 HUD が表示されます。開発・検証目的のための機能です。

### 18-3. フローアニメのフォールバック

URL に `?canvasflow=0` を追加すると、フローアニメを従来の SVG 方式に切り替えられます（トラブルシューティング用）。

## 19. よくある質問

### Q. 図面データはどこに保存されますか？

A. **ブラウザのローカルストレージ**に保存されます。サーバーへは送信されません。ブラウザのキャッシュを削除するとデータが失われるため、重要なデータは「保存」ボタンで JSON ファイルとしてバックアップしてください。

### Q. PDF 背景を読み込むにはどうすればよいですか？

A. 設定パネル（⚙ ボタン）を開き、「PDF 設定」セクションの「PDF 読み込み」ボタンをクリックしてファイルを選択します。オンラインの PDF を使う場合は PDF URL 欄に URL を貼り付けてください。

### Q. 図面を他の人と共有する方法は？

A. 設定パネルの「🔗 共有リンク」ボタンで生成される URL を相手に送ります。URL を開くだけで同じ図面が表示されます。または「保存」で JSON ファイルをエクスポートして送付し、受信者は「読込」で開くこともできます。

### Q. スマートフォン・タブレットでも使えますか？

A. はい。タッチ操作に対応しています。ただし、画面サイズが小さいため、タブレット以上の画面での利用を推奨します。PWA としてホーム画面に追加するとより使いやすくなります。

### Q. オフラインでも使えますか？

A. 一度ブラウザで開いてキャッシュされた後は、インターネット接続なしでも動作します（PWA の機能により）。ただし、最初の読み込み時はインターネット接続が必要です。

### Q. バルブを操作したら図面の流れが変わらなくなりました。

A. まず `Esc` キーを押してすべてのモードをリセットし、操作ツール（V）を選択した状態でバルブをクリックしてみてください。それでも解決しない場合は、ページをリロードして自動保存データから再開してください（自動保存が OFF の場合はデータが失われるため、先に「保存」してください）。

### Q. ノードの連番が飛んでいます。整理できますか？

A. 設定パネルの「連番再振」ボタンで、全ノードの ID を種別ごとに連番で振り直せます（例: V-001, V-002, T-001, T-002...）。

### Q. 操作マニュアルを印刷したい。

A. 設定パネルの「📋 操作マニュアル」を開き、「🖨 印刷設定」で帳票体裁を設定した後、「👁 プレビュー」で確認してから「印刷」ボタンを押してください。

### Q. 図面が重くて動作が遅い場合は？

A. STEAM-BOY は自動的にパフォーマンス最適化を行いますが、以下の点を試してください:
- 使用していないノードや配管を削除して図面を軽量化する
- ブラウザを最新版に更新する
- 他のタブやアプリを閉じてメモリを確保する
- `?canvasflow=0` を URL に追加してフローアニメの方式を切り替えてみる

---

> **困ったことがあれば、設定パネルの「📘 使い方」ボタンでチュートリアルをご覧ください。**
